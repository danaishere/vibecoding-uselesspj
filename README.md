# 🍹 Personality Cocktail Quiz

> **Vibecoding Alert**: This is a fun, no-framework personality quiz built with vanilla HTML, CSS & JavaScript. Pure vibes, zero dependencies!

Answer 5 quick questions. Get a personality cocktail that matches your vibe. That's it!

## How It Works

**Step 1**: Answer personality questions (2 choices each)  
**Step 2**: Each answer gets a point value (0-3 points)  
**Step 3**: Your total score (0-10) unlocks one of 10 unique drinks  
**Step 4**: See your drink + GIF + personality insight

## Quick Start

Open `index.html` in your browser. That's all!

### Files
- `index.html` - Quiz structure
- `style.css` - Styling & animations
- `script.js` - Quiz logic + GIPHY integration

## Customize

**Change Questions** → Edit `quizData` in `script.js`  
**Change Drinks** → Edit `personalityDrinks` in `script.js`  
**Change Colors** → Update `background` gradient in `style.css`  
**Add GIPHY Key** → Replace `GIPHY_API_KEY` in `script.js`
